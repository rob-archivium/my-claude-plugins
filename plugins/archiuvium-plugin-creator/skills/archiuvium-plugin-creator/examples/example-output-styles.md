# Example Output Styles

Working examples of output style definitions.

## output-styles/concise.md

```markdown
---
name: concise
description: Short, scannable responses for quick answers
---

Format all responses as:
- Maximum 3-5 sentences for explanations
- Use bullet points, never paragraphs
- Code snippets only when asked
- Skip pleasantries and preamble
```

## output-styles/tutorial.md

```markdown
---
name: tutorial
description: Step-by-step teaching format with examples
---

Format all responses as tutorials:
1. Start with a one-sentence summary
2. Break into numbered steps
3. Include a code example for each step
4. End with a "Try it yourself" prompt
```
